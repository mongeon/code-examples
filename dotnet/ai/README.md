# .NET AI Examples

AI-related .NET projects integrating local LLMs, embeddings, and vector databases.

Current projects:
- [local-rag-ollama-litellm](local-rag-ollama-litellm): Local Retrieval-Augmented Generation using Ollama, LiteLLM, and Qdrant.

Usage:
- See the project README for setup and commands.
- Typical flow: `dotnet restore` → `dotnet build` → `dotnet run` (inside the project).
