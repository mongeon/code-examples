# .NET Code Examples

This directory hosts .NET projects and snippets organized by topic.

- ai: AI-focused .NET examples (local LLMs, RAG, vector search)

Getting started:
- Navigate to a project folder (e.g., `ai/local-rag-ollama-litellm`).
- Restore and build: `dotnet restore`, `dotnet build`.
- Run console apps: `dotnet run` (inside the app folder).
