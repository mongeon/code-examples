# Azure Static Web Apps avec Blazor - Exemples de code

Exemples de code pour la série d'articles [Azure Static Web Apps avec Blazor](https://www.gabrielmongeon.ca/series/azure-static-web-apps-avec-blazor/) sur [gabrielmongeon.ca](https://www.gabrielmongeon.ca).

## Structure

| Dossier | Article |
|---------|---------|
| `01-getting-started` | C'est quoi une Azure Static Web App? |
| `02-configuration` | Le fichier staticwebapp.config.json |
| `03-api-functions` | Brancher une API Azure Functions |
| `04-authentication` | L'authentification intégrée de SWA |
| `05-local-dev-cli` | Développer localement avec le CLI swa |
| `06-preview-environments` | Les preview environments sur les PR |

## Prérequis

- [.NET 10 SDK](https://dotnet.microsoft.com/download/dotnet/10.0)
- [Azure Functions Core Tools v4](https://learn.microsoft.com/en-us/azure/azure-functions/functions-run-tools-vs-code)
- [Static Web Apps CLI](https://azure.github.io/static-web-apps-cli/)
- Un compte Azure (le tier gratuit suffit pour la plupart des exemples)
- Un compte GitHub

Chaque dossier contient un projet fonctionnel qui peut être déployé indépendamment sur Azure Static Web Apps.
